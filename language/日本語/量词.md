# [量词](https://zhuanlan.zhihu.com/p/21559633)（すうじ）

## 【個（ko）】

用于比较立体的小东西，例如：

1. **一個ケーキ** （一个蛋糕）
2. **二個りんご** （两个苹果）
3. **三個箱** （三个盒子）

## 【つ】

【個】能用的【つ】都能用

1. 表示数量单位的词，只有十个，十个以上直接用数字
2. 可用于抽象事物，比如：
   1. **ひとつお願い。**　一个愿望
   2. **二つ想い。**　两个想法
   3. **八つ歳** 。 8 岁（10 岁以下）
3. 用于计算非扁平、非细长的物体，比上述的【個】更大一些的东西，特别是立体的物体，以及抽象事物。
4. 如果其他的量词都想不起来的话用这个都可以，日本人很喜欢用这个来代替其他量词。

## 【本（ほん）】

用来计算「细长」的东西，只要是「细长」的都可以用

例如：
1. 一本剣　（一把剑）
2. 二本傘　（俩把伞）
3. 三本塔　（三座塔）
4. 四本ビール　（四瓶啤酒）
5. 五本鉛筆 （五根铅笔）
